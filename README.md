## ITZBOT
BOT WHATSAPP YANG BISA DIGUNAKAN DI TERMUX







## CARA INSTALL
# TERMUX
```bash
> download termux
> buka
> pkg install git
> pkg install ffmpeg
> pkg install nodejs
> apt update && apt upgrade
> git clone https://github.com/ItzKunzGaming/itz-bot
> cd itz-bot
> npm i -g cwebp && npm i -g ytdl
> npm i && npm i got
> bash install.sh
> node index.js
```
